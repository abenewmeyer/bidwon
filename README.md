# BidWon - AI-Powered SAM.gov Bid Assistant

## Overview
BidWon helps small and certified contractors automatically discover eligible federal opportunities on SAM.gov, score them against specific NAICS codes, and generate bid packages locked to Llama 3-70B architecture. 

## Current Status
- Modern, high-converting landing page with interactive NAICS demo (Live)
- Supabase backend schema with pgvector (Live)
- Nightly SAM.gov automated sync (Live)
- Stripe Payment Link Integration (Ready)

## Tech Stack
- Frontend: Next.js 15 + TypeScript + Tailwind CSS
- Backend: Supabase (Postgres + Auth + pgvector)
- AI Architecture: Llama 3-70B strictly enforced for compliant generation
- Deployment: Vercel
